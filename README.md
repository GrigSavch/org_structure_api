# Департаменты организации API

REST API для управления организационной структурой компании с поддержкой иерархических подразделений и сотрудников.

> Автор: Григорий Савченко

## Технологии

- **Python 3.11+**
**Django REST Framework** - создание REST API
- **PostgreSQL** - база данных
- **Docker & docker-compose** - контейнеризация
- **drf-spectacular** - OpenAPI документация

## Возможности

- Создание и управление подразделениями (вложенная структура)
- Создание сотрудников внутри подразделений
- Получение детальной информации о подразделении с вложенными детьми (до 5 уровня вложенности)
- Перемещение подразделений (изменение родителя)
- Удаление подразделений с опциями cascade или reassign
- Предотвращение циклических ссылок
- Уникальность названий подразделений в пределах одного родителя
- Валидация входных данных
- Полное логирование операций
- Docker-контейнеризация
- Swagger/OpenAPI документация

## Установка и запуск

### Предварительные требования

- Python 3.11 или выше
- PostgreSQL 15 (для локального запуска)
- Docker и docker-compose (для запуска в контейнерах)
- Git

### Локальный запуск

#### 1. Клонируем репозиторий

```bash
git clone https://github.com/GrigSavch/org_structure_api.git
```

#### 2. Создаем и активируем виртуальное окружение

```
python3 -m venv venv
source venv/bin/activate
```

#### 3. Создаем и активируем виртуальное окружение

```
pip install -r requirements.txt
```

#### 4. Создаем и применяем миграциии

```
python manage.py makemigrations
python manage.py migrate
```

#### 4. Запускаем сервер разработки

```
python manage.py runserver
```
